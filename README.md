# TowerDefense
Des jeux de Strategie.
Pour configurer le jeu sur ta machine rien de compliqur avec python3.8.10
Vous avez besoin juste pip3 et pygame.

Installez pip3 en entrant la commande suivante sur le projet :
sudo apt-get install python3-pip

Une fois l'installation terminée, vérifiez que pip3 est correctement installé en entrant la commande suivante :
pip3 --version

Installez Pygame en entrant la commande suivante :
sudo apt-get install python3-pygame

windows
Assurez-vous d'avoir Python 3.8.10 installé sur votre machine. Si ce n'est pas le cas, vous pouvez le télécharger à partir du site officiel de Python et suivre les instructions d'installation.
Ouvrez une invite de commande en appuyant sur la touche Windows + R, tapez "cmd" dans la boîte de dialogue, puis appuyez sur Entrée. Cela ouvrira l'invite de commande.

Installez pip en entrant la commande suivante :
python -m ensurepip --upgrade

Vérifiez que pip est correctement installé en entrant la commande suivante :
pip --version

Installez Pygame en entrant la commande suivante :
pip install pygame
